# プライバシ-ポリシー  

2026年3月19日

## 1. 収集しない情報について

当拡張機能「Markdown Web Clipper」は、ユーザーのプライバシーを尊重します。当拡張機能がユーザーの個人データ、ブラウジング履歴、またはクリップしたコンテンツを収集、保存、送信、共有することはありません。

## 2. ローカル処理について

メインコンテンツの抽出やMarkdown形式への変換を含む、当拡張機能のすべての機能は、ユーザーのデバイス（ブラウザ内）で完全にローカルに処理されます。データが外部サーバーや第三者に送信されることはありません。

## 3. 権限について

当拡張機能は、コア機能の実現のためだけに以下の権限を要求します：

- `activeTab`: ユーザーが明示的に拡張機能アイコンをクリック、またはショートカットを使用した時にのみ、現在のタブにアクセスするため。
- `scripting`: ページコンテンツを抽出してMarkdownに変換するために必要なスクリプトを注入するため。
- `clipboardWrite`: 生成されたMarkdownテキストをクリップボードに直接コピーするため。
- `storage`: 拡張機能の動作に関するユーザーの設定を保存するため（例：Markdown出力に画像を含めるかどうか）。このデータはローカルに保存され、共有または送信されることはありません。
